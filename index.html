Sistem Informasi Absen dan Lembur Pegawai

  • Description: Website ini dibuat sebagai fasilitas pegawai untuk melakukan absensi, baik absen datang maupun pulang. CovSegNet (Covid-19 Lung Infections Segmentation)
  • Technologies: Python, Tensorflow Keras, Pandas, Flask
  • Description: Website ini dibuat sebagai implementasi dari penelitian terkait segmentasi infeksi paru-paru penderita Covid-19. Penelitian tersebut dilakukan dengan metode Fully Convolutional Neural Network bernama SegNet. Implementasi yang dilakukan ialah melakukan segmentasi terhadap gambar CT-Scan paru-paru penderita Covid-19. Hasil segmentasi berupa gambar yang tersegmen menjadi tiga warna: Abu-abu(Mewakili daerah infeksi), biru(Mewakili Paru-paru bagian kiri), dan kuning(Mewakili paru-paru bagian kanan).

Website Aju Lembur
  • Technologies: PHP (Laravel), Bootstrap 5, MySQL
  • Description: Website ini dibuat sebagai fasilitas pegawai untuk pengajuan lembur. Pegawai dapat mengajukan lembur dengan menyertai alasan. Pengajuan lembur yang baru diajukan akan disimpan dengan status "Diproses". Selama status tersebut, pegawai dapat mengedit dan menghapus pengajuannya. Data pengajuan lembur akan dilakukan review admin dan dilakukan persetujuan atau penolakan.
